Project Summary



  This project examines weather patterns based on distance to the equator. As expected, the weather becomes significantly warmer as one approaches the equator (0 Deg. Latitude). More interestingly, however, is the fact that the southern hemisphere tends to be warmer this time of year than the northern hemisphere. This may be due to the tilt of the earth.

We pulled more than 500 cities data from OpenWeatherMap API. We used the citipy python library to randomly generate coordinates to get the nearest city. Then Using Matplotlib, we took each city's max temperature, humidity, cloudiness, and wind speed and plotted these against latitude in a scatter plot graphs.
